# 🗄️ SQL Practice Queries

## 📌 Overview

This document contains a collection of SQL queries created as part of practice with a simple **Hotel Management Database**.

The goal of this work is to demonstrate understanding of:

* Database structure creation
* CRUD operations
* Filtering and sorting data
* Aggregation functions
* Joins and relationships between tables

## 🧱 Database Structure

The database includes the following tables:

* **Visitors**
* **Rooms**
* **Bookings**

Relationships:

* A visitor can have multiple bookings
* A room can be booked multiple times
* Bookings connect visitors and rooms

## 🧪 Covered SQL Topics

### 🔹 DDL (Data Definition Language)

* CREATE TABLE statements for all entities

### 🔹 DML (Data Manipulation Language)

* INSERT (adding new visitors)
* UPDATE (modifying email)
* DELETE (removing records)

### 🔹 Filtering & Conditions

* WHERE
* AND / OR
* BETWEEN
* LIKE

### 🔹 Sorting & Limiting

* ORDER BY
* TOP (limiting results)

### 🔹 Aggregation Functions

* MIN
* MAX
* COUNT
* SUM
* AVG

### 🔹 GROUP BY

* Grouping rooms by type with average price

### 🔹 JOIN Operations

* INNER JOIN (visitors + bookings + rooms)
* LEFT JOIN (all visitors with bookings)
* RIGHT JOIN (all bookings with visitors)
