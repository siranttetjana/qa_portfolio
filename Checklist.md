# 📋 Registration Form Testing Checklist

## 📌 Overview

This document contains a checklist for testing the **User Registration Form**.
It covers UI, validation, functional behavior, and basic security scenarios.

## 🧪 Scope

* UI / Layout validation
* Input field validation
* Negative and edge case testing
* Successful registration flow
* Basic backend and security checks
