# 🧠 Test Design Techniques

## 📌 Overview

This document demonstrates the application of common **test design techniques** used in QA to create effective and optimized test cases.

The goal is to:

* Reduce the number of test cases
* Increase test coverage
* Identify edge cases and potential defects

---

## 🔹 Test design techniques

* Equivalence Partitioning
* Boundary values
* Decision Table
* Pairwaise testing
* State Transition diagram
* Error Guessing




---

## 📝 Notes

* Techniques can be combined for better coverage
* They help design efficient and meaningful test cases
* Widely used in both manual and automated testing
